##  문자열 내림차순으로 배치하기

### Tip
* join()  
  "-".join(['2010','05','03'])  => '2010-05-03'
  
* split()  
  "2010/05/03".split("/")  => ['2010','05','03']

* sorted()  
  sorted("acbd", reverse = False) => ['a','b','c','d']  
  sorted("acbd", reverse = True) => ['d','c','b','a'] 

**코드에 큰 어려움이 없어 설명을 달지 않음**
